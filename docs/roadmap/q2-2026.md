## Riscos

| Risco | Impacto | Mitigacao |
|-------|---------|-----------|
| API Conta Azul instavel ou lenta | Bloqueia F0 (pagamento) | Testar integracao na S4, ter fallback manual |
| Compliance de horarios incorreto | Cobranca em horario proibido = reclamacao | Testes exaustivos com feriados/horarios |
| Volume do piloto pequeno demais | Nao valida o produto | Buscar 2o piloto em paralelo |
| Imprevistos consomem buffer | Atrasa marcos | 32% de margem ja incluido |
| Bot escala mal para humano | Cliente irritado sem atendimento | Chatwoot como fallback seguro |
